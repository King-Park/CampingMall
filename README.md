# 1. 프로젝트 주제 및 기획의도
> **프로젝트 주제** 

     Spring boot기반의 다양한 API를 활용한 캠핑장 종합 예약 웹 서비스 구축
<br>

> **기획의도/차별점** 

    1. 회원 및 예약과 결제 서비스가 활용되면서 유저(이용 고객)와 관리자(캠핑장 업체) 모두 사용하는 웹 서비스인 캠핑장 종합
       예약 웹 서비스로 기획
    2. 다양한 API 활용과 반응형 웹, 직관적인 UI로 사용자 편의 제공
    3. 캠핑장 업체 관리자의 넓은 웹 컨텐츠 수정 권한 제공
    4. 언제든지 캠핑장 업체가 추가될 수 있는 종합 웹 서비스로 설계
    5. 
    6. 차별성이나 강조할거 또 뭐있지...
<br>

# 2. 프로젝트 목적

    1. 팀 프로젝트 과정으로 협업에 필요한 여러 툴을 사용한 프로젝트 완성
    2. 교육받은 내용을 바탕으로 팀원 모두 프론트/백을 설계, 개발하며 프로세스 이해도 향상 
    3. 네이버 클라우드 플랫폼(NCP)을 사용한 배포를 통해 도메인, 서버에 대한 이해도 향상 및 서버 운영, 유지 보수 경험 축적
    4. 아임포트, kakao, naver, google 등의 다양한 API를 이용하여 예약/결제 및 소셜 로그인, 카카오맵, 날씨 기능, 챗봇 등 구현
    5. Spring boot OAuth2 방식 로그인 인증 구현
    6.
    7. 추가및수정예정
