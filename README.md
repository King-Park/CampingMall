# **1. 프로젝트 주제 및 목적**
> **주제** 

    다양한 API(지도,결제,본인인증,SNS로그인 등)를 활용한 종합 캠핑장 예약 시스템 구축

> **목적**

    1. 고객사로부터 웹사이트 프로젝트 의뢰를 받고 팀과 협업 과정을 통해 의뢰받은 프로젝트 완성 목표
    2. 타 캠핑장사이트와 차별점으로 처음캠핑장을 찾으려는 이용자들도 접근하기쉽게 깔끔한 UI로 예약,설명 구성
    3. github sourcetree 툴을 활용해 팀원과 소통, 협업 과정 경험
    4. spring boot aouth2 방식 인증 구현
    5. 아임포트, kakao, naver, google API를 이용하여 결제 및 소셜 로그인, 카카오맵, 날씨 기능 구현
    6. 네이버 클라우드 플랫폼(NCP)을 사용한 배포를 통해 도메인, 서버에 대한 이해도 향상 및 서버 운영, 유지 보수 경험 축적
